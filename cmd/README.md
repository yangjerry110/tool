# tool
## cmd-tool
### gin-framework
生成gin-framework 脚手架

### 如何使用
> go install github.com/yangjerry110/tool/cmd/gin-framework@latest

### 安装新应用
> gin-framework new newApp
> 根据提示 input 项目地址
> 生成newApp之后，进入到newApp跟目录，执行 `go mod tidy`
